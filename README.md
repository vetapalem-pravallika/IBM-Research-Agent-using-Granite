# 🧠 Research Agent using IBM Granite

## 📌 Problem Statement
The Research Agent is an AI-powered system designed to assist researchers by automating literature review, summarizing papers, identifying research gaps, and generating structured research reports. It improves research efficiency using Agentic AI and RAG architecture.

---

## 🚀 Project Objective
Build an intelligent Research Agent that can:
- Perform literature summarization
- Extract key findings from research papers
- Identify research gaps
- Generate future research directions
- Provide structured academic reports

---

## 🛠️ Technology Stack (Mandatory)
- LangFlow (for workflow orchestration)
- IBM watsonx.ai
- IBM Granite Models
- Retrieval-Augmented Generation (RAG)
- IBM Cloud

---

## 🧩 LangFlow Components Used
- Chat Input → Accepts user research query  
- IBM watsonx AI Agent → Processes query using Granite models  
- Chat Output → Displays generated research insights  
- File Component → Stores datasets and research documents  

---

## 🏗️ Architecture Overview
The system uses a multi-agent RAG-based architecture:
- User submits research query
- LangFlow routes request to IBM watsonx AI
- Granite model processes and generates insights
- RAG retrieves relevant academic knowledge
- Final structured report is generated

---

## 🤖 Key Features
- Literature Review Automation
- Research Gap Identification
- Key Findings Extraction
- Future Research Suggestions
- Structured Report Generation

---

## ⚙️ How It Works
1. User enters a research topic
2. LangFlow workflow processes input
3. IBM Granite model analyzes data
4. RAG retrieves supporting knowledge
5. AI generates structured research output

---

## 📊 Output Format
- Literature Summary  
- Key Findings  
- Research Gaps  
- Future Research Directions  
- References
